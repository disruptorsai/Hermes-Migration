## Execute Plan Workflow

This workflow triggers the **Superpowers Executing Plans Skill** for batch execution with checkpoints.

### Steps

1. Read the executing-plans skill file at `.agent/skills/superpowers/skills/executing-plans/SKILL.md`
2. Follow the skill instructions **exactly as presented** — this executes the plan in batches with review checkpoints.
3. Launch subagent-driven development, working through each engineering task.
4. Inspect and review work at each checkpoint before continuing forward.
5. Do NOT skip checkpoints. Pause for user review at designated points.
