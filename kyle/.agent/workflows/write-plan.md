## Write Plan Workflow

This workflow triggers the **Superpowers Writing Plans Skill** for creating detailed implementation plans.

### Steps

1. Read the writing-plans skill file at `.agent/skills/superpowers/skills/writing-plans/SKILL.md`
2. Follow the skill instructions **exactly as presented** — this creates a clear, detailed implementation plan with bite-sized tasks.
3. The plan should be clear enough for an enthusiastic junior engineer to follow.
4. Emphasize true RED/GREEN TDD, YAGNI (You Aren't Gonna Need It), and DRY.
5. Present the plan for user approval before proceeding to execution.
