# Pick Niche

## Purpose
Define and validate the target market or niche before research begins.

## Input
- Raw user input describing the niche, market, or topic of interest

## Process
1. Parse the user's description to identify the core market
2. Validate that the niche is specific enough for meaningful research
3. If too broad, narrow it down (e.g., "AI tools" becomes "AI video editing tools for creators")
4. Output a structured niche definition with category, audience, and scope

## Output
- Niche name, target audience, market category, and scope boundaries
