# Social Posts

When writing social posts:

- **X/Twitter**: Short, punchy, opinionated. Use line breaks for rhythm. No hashtags unless asked.
- **LinkedIn**: Professional but human. Lead with a bold statement. Use the "hook → story → lesson" format.
- **Instagram**: Visual-first caption. Shorter is better. Use emojis sparingly.

Always write 2-3 variations so the user can pick.
