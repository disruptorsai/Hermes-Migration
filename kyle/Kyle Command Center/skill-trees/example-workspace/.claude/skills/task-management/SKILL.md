# Task Management

This is a shared skill used by all agents.

When managing tasks:

1. Check the current task list before adding duplicates
2. Each task needs: title, assignee, priority (high/mid/low), status (todo/in-progress/done)
3. Update status as work progresses
4. Flag blocked tasks with a reason
