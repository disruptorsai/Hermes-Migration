# Daily Planning

This is a shared skill used by all agents.

When helping with daily planning:

1. Ask what the user's top 3 priorities are for the day
2. Suggest time blocks based on priority and estimated effort
3. Identify one "big rock" that must get done no matter what
4. Flag anything from yesterday that's still unfinished
