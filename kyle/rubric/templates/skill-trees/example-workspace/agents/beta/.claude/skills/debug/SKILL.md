# Debug

When debugging an issue:

1. Reproduce the bug — confirm you can trigger it
2. Isolate — narrow down to the smallest failing case
3. Read the relevant code, don't guess
4. Fix the root cause, not the symptom
5. Verify the fix doesn't break anything else
6. Explain what was wrong and why the fix works
