# Script Writing

When writing a video script:

1. Start with a hook — the first 5 seconds decide if people stay
2. Structure: Hook → Problem → Solution → Proof → CTA
3. Write for spoken delivery — short sentences, natural rhythm
4. Include visual cues in brackets: [show screen], [cut to B-roll]
5. End with a clear call to action
