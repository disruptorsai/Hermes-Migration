# Write Report

## Purpose
Compile all research findings into a structured, readable market research brief.

## Input
- Niche definition, competitor list, and positioning analysis from previous steps

## Process
1. Structure the brief with standard sections: Overview, Market Landscape, Competitor Profiles, Positioning Map, Gaps and Opportunities
2. Write clear summaries for each section
3. Add a recommendations section highlighting underserved areas
4. Format as clean markdown with tables where appropriate

## Output
- A complete markdown document ready for review and sharing
